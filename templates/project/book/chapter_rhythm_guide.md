# 网文章节节奏参考

> 本文件提供不同网文章类型的正面写作模板。写入 rolling_plan 的 `chapter_function` 字段时，参考对应的节奏链。

## 打脸章 (face_slap)
- **核心爽点**：围观者的认知被主角的实力/身份/手段碾压式逆转。
- **节奏链**：对方质疑/轻视 → 主角低调或无视 → 对方进一步挑衅 → 主角出手（或被动触发实力）→ 碾压式的实力展示 → 对手沉默/众人震惊。
- **压力曲线**：外部质疑堆积 → 主角展示 → 认知反转 → 余波。
- **常见错误**：碾压后主角长篇大论解释自己多强（破坏爽感）；"众人震惊"只用一句带过（浪费最核心的情绪输出窗口）；对手太弱导致无成就感。
- **叙事织入重点**：围观者反应要分层次写——近处的、远处的、之前看不起主角的、一直中立的、之前就相信主角的。

## 升级/突破章 (breakthrough)
- **核心爽点**：克服瓶颈后的力量质变感和新能力展示。
- **节奏链**：遭遇瓶颈 → 尝试常规方法 → 失败 → 机缘/顿悟/积累触发 → 突破临界 → 身体/力量变化 → 小测试验证新力量。
- **压力曲线**：内部压力累积 → 临界点 → 突破释放 → 力量验证。
- **常见错误**：突破过程写成纯内视/纯说明（缺少外部世界反馈）；突破后不测试力量直接切章（读者没看到新能力的效果）；每次突破的写法雷同。
- **叙事织入重点**：突破瞬间的环境变化（灵气波动、天地异象）、旁观者的感知（如果有人在场）、突破后的身体感受变化。

## 拍卖/交易章 (auction)
- **核心爽点**：宝物归属的悬念、价格博弈的刺激、主角捡漏或用财力碾压。
- **节奏链**：宝物展示 → 竞价开始 → 意外竞争者 → 价格推高 → 主角出手（捡漏/碾压/放弃但另有收获）→ 拍后交锋或跟踪。
- **压力曲线**：预期 → 竞价拉锯 → 意外出现 → 结果落地 → 后续暗流。
- **常见错误**：拍卖品罗列过多变成清单；每次拍卖都是主角买最贵的；没有拍卖后的竞争者反应。
- **叙事织入重点**：拍卖场的环境质感、其他竞拍者的微表情和动机暗示、拍卖师的控场节奏。

## 赶路/探索章 (travel)
- **核心爽点**：新地图的初见感、路途中的小遭遇、世界质感的自然展开。
- **节奏链**：出发 + 环境引入 → 路途中的小遭遇或世界细节 → 信息碎片（传说/见闻/遗迹线索）→ 到达或发现。
- **压力曲线**：低开 → 遭遇小波动 → 信息浮现 → 到达或停留。
- **常见错误**：赶路写成纯粹的地理移动（没有遭遇、没有质感）；用一句话跳过整个移动过程（浪费世界展示机会）。
- **叙事织入重点**：天气、地貌、路人的日常、食物、住宿、货币、口音——这些是让世界可信的材料。

## 种田/经营章 (building)
- **核心爽点**：投入产出的确定性反馈——每一份努力都有可见的收获。积累感和规模扩张的快感。
- **节奏链**：小投入 → 小收获 → 扩大规模/复制经验 → 遇到外部阻力 → 解决或绕开 → 阶段性成果 → 发现新机会。
- **压力曲线**：低开 → 渐进积累 → 外部干扰出现 → 守住成果。
- **常见错误**：只写经营数据不写人物反应（变成报表）；阻力永远能被轻松解决（缺少真实感）；经营和主线完全割裂。
- **叙事织入重点**：经营过程中的人物互动、雇工/村民/属下的反应、资源从紧缺到充裕的心理变化、经营细节中的世界质感。

## 副本/规则章 (dungeon_rule)
- **核心爽点**：规则探索的智力快感——发现规则、利用规则、突破规则。
- **节奏链**：进入新环境 → 观察/试探规则 → 规则反噬或意外 → 找到漏洞或规律 → 利用规则通关或付出代价退场。
- **压力曲线**：未知 → 试探压力 → 反噬高压 → 理解后的控制感或代价。
- **常见错误**：规则只靠旁白解释而不用事件验证；主角一眼看穿所有规则（缺少试错过程）；通关后没有成本或代价。
- **叙事织入重点**：规则触发时的物理/感官反馈、队友或同场者的不同反应、规则本身的"不合理"之处（为后续伏笔）。

## 权谋试探章 (political_probe)
- **核心爽点**：信息不对称下的智力博弈——对方出招、主角解读、假装不知、反制布局。
- **节奏链**：对方出招（信息试探/威胁暗示）→ 主角解读隐含信息 → 假装不知或假装中计 → 暗中反制 → 对方反应（是否察觉被反制）。
- **压力曲线**：外部试探抵达 → 表面配合/内部解读 → 暗中布局 → 双方认知差拉大。
- **常见错误**：试探内容全部写成对话（缺少非语言信息——表情、站位、时间选择）；主角每次都能完美应对；对手的试探方式千篇一律。
- **叙事织入重点**：试探场景的环境选择（为什么在这里说）、在场者的不同身份和立场、说话人和听话人之间的"第三层意思"。

## 关系拉扯章 (relationship)
- **核心爽点**：感情张力的涨落——靠近又拉开、误解又澄清、心动又克制。
- **节奏链**：靠近（互动/关心）→ 误解或外部伤害 → 情感拉开距离 → 一方主动靠近或双方僵持 → 关系发生质变（确认/破裂/新的默契）。
- **压力曲线**：温暖靠近 → 冲突或伤害 → 冷淡拉远 → 主动破冰或继续僵持。
- **常见错误**：误会全靠不沟通推动（人物降智）；每次关系波动都用同一个模式；甜的时候全是甜、虐的时候全是虐（缺少复杂情绪）。
- **叙事织入重点**：非语言的感情信号（眼神、手势、距离、停顿）、身体反应（心跳、呼吸、温度）、日常物件的情感承载。

## 余波/回收章 (aftermath)
- **核心爽点**：重大事件后的世界反馈——"这件事改变了什么"。也是伏笔回收的黄金窗口。
- **节奏链**：事件后果在各层面显现（个人/关系/势力/世界）→ 人物消化和反应 → 新常态建立 → 下一个事件的萌芽出现。
- **压力曲线**：高开（事件刚结束）→ 回落消化 → 新常态 → 下一条线索浮现。
- **常见错误**：余波章写成纯总结（变成"前情提要"而不是新剧情）；所有层面用一段带过（个人/关系/世界的变化要分别落地）；下一事件的萌芽太隐晦或太明显。
- **叙事织入重点**：不同人物对同一事件的不同理解、媒体/旁观者/势力的反应如何塑造"公开版本"、日常生活的恢复与变化的不可逆。
