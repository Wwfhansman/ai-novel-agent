# 文笔风格报告生成提示词

> 使用方式：把下面整段提示词复制给 AI，同时上传或粘贴你的对标小说 TXT。让 AI 按提示词输出一份「文笔风格模仿报告」。拿到报告后，把报告复制到 `style/samples.md`。
>
> 注意：`sample_prompt.md` 是工具模板，不应该作为写作风格样本读入 writing packet。真正给写作 agent 参考的是 `style/samples.md`。

## 可复制提示词开始

你是一名小说文笔分析师。请阅读我提供的对标小说 TXT，输出一份可供 AI 写作 agent 使用的「文笔风格模仿报告」。

目标不是复刻原作内容，而是提取可迁移的语言方法，让另一本完全不同的小说在句子节奏、段落手感、描写温度、对话语感和情绪处理上接近目标文本。

只分析文笔，不分析剧情结构，不复述故事，不总结人物关系，不提炼设定体系。

请严格遵守：

1. 不复述完整剧情。
2. 不输出大段原文。
3. 单个原文短摘录最多 25 个字，只能用于说明文笔特征。
4. 不提取、不推荐复用原作的人名、地名、势力名、特殊物品、核心设定名、标志性桥段。
5. 不生成“续写原作”或“仿写原作剧情”的内容。
6. 可以生成完全原创、无关原作剧情的小段落，用来示范这种文笔如何迁移到其他故事。
7. 输出内容要能直接粘贴进小说项目的 `style/samples.md`，供后续 AI agent 参考。

## 分析方法

请先完成抽样校准，再写报告：

- 从文本开头、中段、高潮/冲突段、低压过渡段、对话密集段各取样观察。
- 如果文本很长，不需要通读每一字，但必须覆盖不同场景密度和情绪强度。
- 区分“作者稳定习惯”和“某一场景临时需要”。报告只写稳定习惯；临时手法标记为“场景限定”。
- 每个结论尽量落成写作规则，不要只写审美形容词。
- 如果某项判断不确定，写“不稳定/样本不足”，不要强行总结。

### 可以增强

列 5-8 条。能提高相似度，但不能过量。

### 谨慎使用

列 5-8 条。容易造成抄袭感、油腻感、AI 腔或风格过拟合。

## 叙述腔调

- 叙述者离人物近还是远
