# 文笔风格报告生成提示词

> 使用方式：把下面整段提示词复制给 AI，同时上传或粘贴你的对标小说 TXT。让 AI 按提示词输出一份「文笔风格模仿报告」。拿到报告后，把报告复制到 `style/samples.md`。
>
> 注意：`sample_prompt.md` 是工具模板，不应该作为写作风格样本读入 context pack。真正给写作 agent 参考的是 `style/samples.md`。

## 可复制提示词开始

你是一名小说文笔分析师。请阅读我提供的对标小说 TXT，输出一份可供 AI 写作 agent 使用的「文笔风格模仿报告」。

重点只分析文笔，不分析剧情结构，不复述故事，不总结人物关系，不提炼设定体系。

我要学习的是：

- 语言质感
- 句子长短和节奏
- 段落手感
- 叙述腔调
- 描写方式
- 对话语感
- 情绪表达方式
- 修辞偏好
- 留白方式
- 画面密度
- 文字的冷暖、轻重、快慢

请严格遵守：

1. 不要复述完整剧情。
2. 不要输出大段原文。
3. 单个原文短摘录最多 25 个字，只能用于说明文笔特征。
4. 不要提取或推荐复用原作的人名、地名、势力名、特殊物品、核心设定名、标志性桥段。
5. 不要生成“续写原作”或“仿写原作剧情”的内容。
6. 可以生成原创的、无关原作剧情的小段落，用来示范这种文笔如何迁移到其他故事。
7. 输出内容要能直接粘贴进小说项目的 `style/samples.md`，供后续 AI agent 参考。

请按以下结构输出：

```markdown
# 文笔风格模仿报告

## 来源说明

- 对标文本：用户提供的 TXT
- 用途：仅作文笔风格参考
- 禁止迁移：原作人物、剧情、世界观、专有名词、标志性桥段
- 允许迁移：语气、句法、节奏、描写方法、段落手感、情绪表达方式

## 一句话文笔定位

用 1-3 句话概括这本小说的文笔气质。

不要写剧情类型。请写语言感受，例如：

- 冷静、细密、克制
- 明快、利落、带锋芒
- 绵密、感官强、情绪浓
- 白描为主，偶尔用重句落点
- 句子短，动作密，压迫感强
- 叙述温柔但有暗刺

## 文笔关键词

列出 10-20 个关键词，描述语言本身。

每个关键词后解释它在文本中如何体现。

## 叙述腔调

分析这本小说的叙述声音。

请回答：

- 叙述者离人物近还是远？
- 叙述是冷眼旁观，还是贴着人物感受？
- 是否带讽刺、怜悯、温柔、压迫、幽默、疏离？
- 是否经常替人物解释心理？
- 是否更喜欢让读者自己从动作和物件里感受？
- 叙述声音稳定吗？高潮时是否变快或变重？

输出可模仿规则：

- 叙述距离：
- 叙述温度：
- 叙述者该说什么：
- 叙述者不该说什么：

## 句子节奏

重点分析句子的长短、快慢、顿挫。

请具体说明：

- 平均句子偏长还是偏短
- 长句通常用在什么地方
- 短句通常用在什么地方
- 是否常用连续短句制造压迫
- 是否常用长句包裹复杂感受
- 是否喜欢一句话末尾突然落重音

- 节奏模板 1：
- 节奏模板 2：
- 节奏模板 3：
- 不要模仿过度的地方：

## 段落手感

只分析分段和段落内部节奏。

请说明：

- 多数段落是一句话、两三句话，还是长段
- 对话是否单独成段
- 动作是否单独成段
- 心理活动是否拆段
- 描写和判断是否混在一段
- 手机上读起来是否轻
- 哪些段落最有呼吸感

输出适合本项目的分段规则：

- 多数段落建议长度：
- 什么情况下必须另起一段：
- 哪种长段可以保留：
- 哪种长段必须拆：
- 短段怎么写才有力量：
- 短段怎么写会显得空：

## 用词偏好

分析用词层面：

- 词语偏口语还是书面
- 是否偏古典、现代、冷硬、柔软、华丽、朴素
- 动词是否精确
- 形容词多不多
- 副词多不多
- 是否常用颜色、气味、触感、声音
- 是否喜欢抽象词，例如命运、规则、秩序、荒唐、体面
- 是否喜欢具体物件，例如杯盏、衣料、门槛、灯影

## 描写方法

重点分析“怎么描写”，不要分析“描写了什么剧情”。

### 人物描写

- 外貌是否细写
- 是否用动作代替外貌
- 是否用衣着、手、眼神、站姿、声音建立人物
- 是否一次写全，还是只给一两个记忆点

### 场景描写

- 场景是铺开写，还是只抓几个物件
- 是否通过光线、声音、气味、温度建立氛围
- 是否用环境反衬人物状态

### 动作描写

- 动作是否连续
- 是否写微动作
- 动作是否承担心理和信息
- 动作句是否短

### 心理描写

- 是否直接写“他/她想”
- 是否更喜欢用动作、停顿、视线、物件替代
- 内心判断是否冷静、锋利、混乱、抒情

- 应避免的误用：

- 意象偏冷、暖、暗、亮、湿、硬、轻、重

- 如何避免写成堆砌：

请说明：

- 情绪是直接命名，还是通过动作/物件/环境呈现
- 悲伤、愤怒、紧张、心动、恐惧分别如何落笔
- 是否压抑情绪，不让人物喊出来
- 是否通过身体反应写情绪
- 是否通过句子变短/变长写情绪

请说明：

- 是否每段都有实物
- 是否有大量抽象判断

- 画面是否清楚但不堆
```
